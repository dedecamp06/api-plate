## Installation  
Dentro da pasta do projeto rode 
```npx sequelize db:migrate```
Apos isso abra o projeto para criar as tabelas.

Depois use o comando 
```npx sequelize-cli db:seed:all ``` para criar as seeders
```
Using npm:
```bash
$ npm install
```

Using yarn:
```bash
$ yarn
```
## Running the app

```bash
# development
$ npm start

```

Using docker:
```bash
$ docker build -t name/node
```
